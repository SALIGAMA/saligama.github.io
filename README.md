# Harish Saligama Resume Site

Static resume site prepared for GitHub Pages.

## Files

- `index.html`
- `styles.css`
- `assets/photo.jpg`
- `.nojekyll`

## Publish on GitHub Pages

1. Create a public repository on GitHub.
2. For the cleanest URL, name it `saligama.github.io`.
3. Upload all files from this folder to the repository root.
4. In GitHub, open `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/(root)`.
7. Wait a few minutes for the site to go live.

If the repository is named `saligama.github.io`, the public URL will be:

`https://saligama.github.io/`
